# TypeScript

# 1.什么是ts?

ts是js的超集，扩展了js的语法，最终都会编译成js

# 2.为什么使用ts?

ts在开发时就能给出编译错误，js要在运行时才能给出错误

# 3.什么是泛型？

泛型指定义函数，接口或类的时候，不预先指定具体的类型，使用时再去指定类型

function createArray<T>(length: number, value: T): Array<T> {
let result: T[] = [];
for (let i = 0; i < length; i++) {
result[i] = value;
}
return result;
}
// 使用的时候再指定类型
let result = createArray<string>(3, 'x');
// 也可以不指定类型，TS 会自动类型推导
let result2 = createArray(3, 'x');
console.log(result);

# 4.什么是构造函数

主要用来创建对象时初始化对象。可以通过this来访问当前类中的属性和方法

# 5.重写是什么？

子类可以根据需要重写父类的方法

# 6.never和void区别

never是所有类型的子类型，表示永远不存在的值 void表示没有任何类型，可以被赋值为null和undefined

# 7.类型和接口的区别

类型可以声明任何类型(包含基本类型，联合类型和交叉类型)，接口只能定义对象类型 接口可以扩展和实现，类型不可以

# 8.联合类型

string|number

# 9.断言 as Foo 用来去覆盖原有的推断

interface Foo {
  bar: number;
  bas: string;
}

const foo = {} as Foo; 去覆盖原有的推断
foo.bar = 123;
foo.bas = 'hello';
