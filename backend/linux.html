    # 常用命令

    cp 原文件名称 新文件名称

    du -h 项目名称 查看文件大小 --max-depth=1 不向下递归

    free 查看服务器内存情况

    rz 上传

    sz 下载

    top 查看cpu情况

    df -h 查看磁盘空间

    tail -n 10 查看最后10行

    more -c -60 每页60行 /+字符进行搜索，按n继续往下搜索

    cat 查看

    find /usr/tmp -name 'a*' 搜索/usr/tmp下以a开头的文件

    uname -r 查看内核版本

    uname -a 查看系统信息

    ps -ef | grep tomcat 查看进程信息

    ps -ef | grep 1095 根据进程id查看进程信息

    cat a.log | grep '2021-08-30' 查看日志里包含2021-08-30的日志

    chmod 777 修改权限 最大权限

    chmod 755 权限范围小点 除拥有者其他人只可读可执行

    tab 命令补全

    which查询命令所在路径

    whereis 查找文件

    where 查找文件位置

    locate -i keyword 忽略大小写查找文件

    find / -name filename查找文件

    gzip 解压

    tar 可压缩和解压
