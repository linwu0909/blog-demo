# HTML

# 1.h5语义化的好处是什么？

结构良好，便于阅读(利于seo优化)。如header，footer，article，aside(都属于块级元素)

# 2.说一下块元素和行元素？

  • 块元素：独占一行，自动填满父元素，可以设置margin,padding以及高度和宽度 (div h1 p)
  • 行内元素：不会独占一行，不可以设置宽高，在垂直方向padding和margin失效 (span input strong)

# 3.img的title和alt有什么区别？

title当鼠标移到元素上会显示，alt是当图片无法加载时会显示

# 4.h5新特性？

  1. 语义化标签
  2. 本地存储localStorage sessionStorage
  3. 新增video和audio元素，常用于富文本

# 5.seo？

  • seo是搜索引擎优化
  • seo比较常见的是通过title，description，keyword来提高搜索权重。
  • title，description，keyword三个权重从高到低
  • 首页title：网站名- 内容页：内容标题-网站名
